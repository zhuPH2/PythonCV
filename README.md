# SeeImagePX
基于python和OpenCV的图形界面下图形鼠标处的像素位置和灰度值等信息显示。



## LICENSE

采用MIT License，详见 LICENSE 文件。



安装以下依赖库：

opencv-python

numpy



## FAQ

### 如何使用
点击菜单栏-打开 打开图片文件。
下方菜单栏将显示该图片鼠标下的像素和灰度值等信息。


### 支持的图片类型
PNG JPG BMP GIF
